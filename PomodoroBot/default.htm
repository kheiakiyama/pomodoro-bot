PomodoroBot

PomodoroBot はポモドーロタイマーを実行する Bot サービスです
現在 Slack, Skype に対応しています



How to use

  1. 上のボタンからあなたが使うチャットサービスのBotを登録します
  2. 登録したチャットサービス上で 
    create
    コマンドを @pomodoro に送ってください
    @pomodoro はタイマーの設定を促します
    設定が完了すると
    [start {id}]
    のような返信が送られます
    なお、設定の途中で操作がわからないときは 
    ?
    コマンドでヘルプを表示できます
  3. 返信のとおり、
    start {id}
    コマンドでタイマーを開始します
  4. タイマーが不要になったときは 
    stop {id}
    コマンドでタイマーを終了します

Command List

  1. create
    タイマーを作成します
  2. delete [id]
    タイマーを削除します
  3. list
    タイマーを一覧表示します
  4. start [id]
    タイマーを開始します
  5. stop [id]
    タイマーを停止します
  6. help
    ヘルプを表示します
